Bienvenue à toi, Artiste !! Tu vas devoir créer un guerrier capable de se battre. Pour ce faire, tu as à ta disposition un projet avec un ring comme décor, un Skeletal Mesh ainsi que diverses animations rangées dans SimFightContent. Les tâches à accomplir sont les suivantes :
- créer le personnage
- utiliser toutes les anims présentes dans le projet
- créer un/des blendspace(s)
- créer un AnimationBlueprint
- créer les contrôles pour que notre guerrier puisse :
  - ne rien faire (idle)
  - marcher
  - reculer (pas se tourner, vraiment reculer)
  - courir
  - s'accroupir
  - marcher accroupi (sneak walk)
  - sauter : 3 étapes -> le saut doit être décomposé en 3 animations : l'impulsion du saut, quand il est en l'air, et quand il atterit.
  - donner un coup de poing (punch)
  - donner un coup de pied (kick)
  - sauter
  
Il est possible que tu n'aies pas toutes les animations nécessaires dans le projet. N'hésite pas à en récupérer sur Mixamo.
De la même façon, tu as accès au Starter Content si tu en as besoin.
